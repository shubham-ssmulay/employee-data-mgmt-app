# Employee Manager Frontend

## Overview
Frontend for Employee Manager using **React**, **Vite**, and **Tailwind CSS**.

---

## Requirements
- Node.js 18+  
- npm or yarn

---

## Directory Structure
```
frontend/
├── src/
│ ├── App.jsx
│ ├── main.jsx
│ └── index.css
├── vite.config.js
└── package.json
```

---

## Install Dependencies
```bash
cd frontend
npm install
```

## Run Frontend
```npm run dev```

Default port: `5173`

Navigate to this URL - 

```
http://localhost:5173/
```
